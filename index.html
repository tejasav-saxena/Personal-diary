Personal Diary Application

Description

This Personal Diary Application allows users to write, save, and load daily diary entries. It's built using tkinter for the GUI, making it a standalone desktop application.

Key Features

  • Enter your username for personalized diary entries.
  • Write multi-line entries using tkinter.Text() widget.
  • Save entries with the current date, organized by username and date.
  • Load previous entries from the same day.
  • Clear text box for new entries each day.

How to Install and Run

To use this application, you need Python installed on your system. Follow these steps:

  1. Clone this repository or download the Python script personal_diary.py from the repository.
  2. Run the following command in your terminal:
    3. python personal_diary.py
